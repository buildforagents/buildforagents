[AgentMail](https://agentmail.to) - Email inboxes for agents
[AgentRQ](https://agentrq.com) - [Github](https://github.com/agentrq/agentrq) - A human-in-the-loop task manager that let you and supervisor agent lead colony of AI agents in multiple projects
