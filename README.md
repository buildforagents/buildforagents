# Build for Agents
A curated list of tools and services that are specifically built for agents

# Tools & Services

- [AgentMail](https://agentmail.to): Email inboxes for agents
- [AgentRQ](https://agentrq.com) - [Github](https://github.com/agentrq/agentrq), [MCP](https://mcp.agentrq.com/mcp), [SVG](./icons/agentrq.svg):  A human-in-the-loop task manager that let you and supervisor agent lead colony of AI agents in multiple projects
